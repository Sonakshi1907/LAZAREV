 # 🚀 LAZAREV AI Agency Website

A modern and visually engaging AI Agency website inspired by Lazarev, built using **HTML, CSS, JavaScript, and GSAP**. The website features smooth animations, interactive UI elements, and a clean, responsive design to showcase AI services and digital products.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/sonakshisoni1907/lazarev-ai-agency.git
````

### Navigate to the Project

```bash
cd lazarev-ai-agency
```

### Run the Project

Simply open the `index.html` file in your browser.

Or use **Live Server** in Visual Studio Code.
